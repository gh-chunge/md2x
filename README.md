# md2x —— 原生js实现markdown语法解析器

- 作者：春哥 <dev@chun.ge>

## 支持格式
### 行间
标题、链接、图片、代码、公式、引用、列表、提示、图表、流程图、时序图、甘特图、饼状图、脚注。

### 行内
加粗、倾斜、下划线、删除线、代码、公式。

### 准备支持的格式
分页、参考文献、表格、多图片切换、颜文字 Emoji、翻译对照、折叠区、锚点等等
